# Markdown Cheatsheet

---

## Headings

```markdown
# H1 — Big title
## H2 — Section
### H3 — Subsection
#### H4 — Smaller
```

---

## Text Formatting

```markdown
**bold**
*italic*
~~strikethrough~~
`inline code`
```

---

## Lists

```markdown
# Unordered
- item
- item
  - nested item

# Ordered
1. first
2. second
3. third
```

---

## Links & Images

```markdown
[link text](https://url.com)
![alt text](image.jpg)
```

---

## Code Blocks

````markdown
```bash
echo "hello"
```

```python
print("hello")
```
````

---

## Tables

```markdown
| Name  | Age |
|-------|-----|
| Alice | 25  |
| Bob   | 30  |
```

---

## Blockquote

```markdown
> This is a quote
> it can span multiple lines
```

---

## Horizontal Line

```markdown
---
```

---

## Checkboxes

```markdown
- [x] done
- [ ] not done
```
